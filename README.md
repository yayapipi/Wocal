# Wocal 打卡系统
Wocal打卡系统是有一天我在学校的新创电动车运营团队的时候，因为需要一个打卡系统而研发出来的，这是个用PHP建构的打卡系统，可以扫描一下卡号后自动在后台打卡记录上下班的时间，这个打卡系统只要简单设定一些参数后就可以套用在任何地方上，所以在这里也把源码公布出来让大家使用。
